# oumuamua - web

## 1.环境

## `node`

## 2. 安装依赖
## `npm install`

## 3. 启动项目

## `npm run dev` 

####  启动失败看报错信息缺少什么模块 在继续`npm install 模块` 然后继续 `npm run dev` 即可


## 4. 代码

| 路径                       | 名称 |
| -                         | - |
| `src/pages/home`          | 首页 |
| `src/pages/capital`       | 资金市场 |
| `src/pages/loan`          | 借贷市场 |
| `src/pages/launchLoan`    | 发起借款 |
| `src/pages/launchCapital` | 发起资金 |
